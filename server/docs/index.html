Saha Platform API v1

Documentation for the Saha Marketplace Platform REST API.

Base URL: /api

Authentication

POST /auth/register

Register a new user.

{
  "email": "user@example.com",
  "password": "securepassword",
  "name": "John Doe"
}
POST /auth/login

Login to receive an access token.

{
  "email": "user@example.com",
  "password": "securepassword"
}
/* Response */
{
  "token": "eyJhbGciOiJIUzI1NiIsIn..."
}

Ads

GET /ads

Get a list of ads. Supports query params for filtering (category, city, etc).

GET /ads/:id

Get details of a specific ad.

POST /ads Auth Required

Create a new ad.

{
  "title": "iPhone 15 Pro",
  "description": "Like new condition",
  "price": 3500,
  "category": "electronics",
  "cityId": "riyadh"
}

Utility

GET /countries

Get list of supported countries.

GET /health

Service health check.

POST /setup-database

Initialize or reset the database schema and seed data.
